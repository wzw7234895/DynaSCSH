# DynaSCSH

This repository contains the reproducibility artifact for the paper:

**Dynamic Size-Constrained Community Search over Evolving Heterogeneous Information Networks**

The repository is prepared for a PVLDB submission / preprint artifact release. Do **not** describe the paper as accepted by PVLDB unless acceptance has been confirmed.

## Artifact Status

- Venue target: PVLDB 2027
- Review mode: PVLDB is single-blind, so author information may appear in the paper.
- Public artifact: this repository is intended to provide a stable public copy of the code, data, paper source, and result files used by the manuscript.
- Paper PDF: `paper/main.pdf`
- Main reproducibility entry point: `experiments/verify_invariants.py`

## Environment

Tested with:

- Python 3.10
- `networkx` 3.4.2
- `numpy` 2.2.6
- `scipy`

Minimal setup:

```bash
python -m venv .venv
source .venv/bin/activate  # Windows PowerShell: .venv\Scripts\Activate.ps1
pip install networkx numpy scipy
```

The paper PDF is already included. Recompiling the LaTeX source requires a standard LaTeX distribution with ACM/PVLDB-compatible packages.

## Quick Verification

Run from the repository root:

```bash
python experiments/verify_invariants.py
```

Expected output:

```text

```

## Reproducing Results

### Synthetic 22-Configuration Benchmark

```bash
python experiments/run_experiments.py --all --output results
```

This regenerates the synthetic result files and `results/results_all_22.csv`. On a CPU-only machine, the full run may take tens of minutes.

### Hybrid Benchmark

```bash
python experiments/run_hybrid.py
```

The hybrid benchmark uses `data/DBLP_processed/` and injects planted dense communities into the processed DBLP topology. The paper tables use the compact summaries:

- `results/hybrid_summary.json`
- `results/hybrid_targeted_repair.json`

## Scope and Limitations

- The implementation maintains communities under projected source-edge insertions and deletions for a single symmetric meta-path.
- Raw HIN relation updates require projection refresh or index rebuild.
- The static baseline and DynaSCSH fallback both preserve the same original query node.
- Deletion Q-Ratio below 1 is intentionally reported as a continuity-quality trade-off, not hidden as a non-degradation guarantee.

## Data Notes

The synthetic DBLP-like benchmark is generated by the experiment code. The hybrid benchmark uses processed DBLP files under `data/DBLP_processed/` and follows the MAGNN-style DBLP schema cited in the paper.
